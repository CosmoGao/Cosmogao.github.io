安装 Windows 8.1

1. 安装 Windows 8.1系统

  • 插入刚刚制作好的U盘
  • 再次检查备份是否完好
  • 重启计算机
  • 在出现启动画面时按F12进入引导选择界面，选择从U盘引导启动
  • 按照提示进行安装操作

2. 激活 Windows 8.1

  • 下载下面的激活工具

    HEU_KMS_Activator_v10.0.0.exe 密码: rew8
    SHA1 6334D3438186A114299F68EDBD819BF9DF1F2766

  • 使用组合键 Win+R 打开运行对话框，输入 slmgr.vbs -xpr 检查激活情况

升级 Windows 10

1. 升级 Windows 10 系统

  • 双击打开之前下载的 Windows 10 镜像
  • 运行 Setup.exe 并按照提示操作
  • 可选不保留任何设置和文件

2. 检查激活情况

  • 使用组合键 Win+R 打开运行对话框，输入 slmgr.vbs -xpr 检查激活情况
  • 一般已激活系统升级至 Windows 10 之后，会自动永久激活
